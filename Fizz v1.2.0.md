## Versiyon Kontrolü Ve Durdurma Komutları : 

#### Versiyon Kontrolü : 

```bash
sphnctl fizz version
```

#### Durdurma : 

```bash
sphnctl fizz stop
```


#### Çalıştığını Anlamak İçin Mini Log Kontrol Komutu : 

```bash
sphnctl fizz logs
```
